    3. Call $this->permissionRepo->assignLevelPermission($levelId, $pageId)
    4. Catch PDOException: if message contains 'foreign key' → return notFoundResponse('Page not found'); if contains 'unique' or 'duplicate' → return errorResponse 409 DUPLICATE_ENTRY 'Permission already assigned'; otherwise rethrow
    5. Return jsonResponse with ['message' => 'Permission assigned'], status 201

    `remove()` method:
    1. Validate level exists → notFoundResponse if null
    2. Parse body, validate page_id is present → validationErrorResponse if missing
    3. Call $this->permissionRepo->removeLevelPermission($levelId, $pageId)
    4. If returns false → return notFoundResponse('Permission not found')
    5. Return jsonResponse with ['message' => 'Permission removed'], status 200

    `getMatrix()` method:
    1. Validate level exists → notFoundResponse if null
    2. Call $this->permissionRepo->getLevelMatrix($levelId)
    3. Return jsonResponse with ['data' => $matrix], status 200
