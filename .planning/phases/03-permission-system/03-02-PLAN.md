    `assignUserPermission(int $userId, int $pageId, bool $isGranted): void`:
    - Use Capsule::table('user_permissions')->updateOrInsert(['user_id' => $userId, 'page_id' => $pageId], ['is_granted' => $isGranted])
    - Catch PDOException: if message contains 'foreign key' → throw new \Imc\Domain\Exceptions\NotFoundException('User or page not found'); if contains 'unique' or 'duplicate' → this shouldn't happen with updateOrInsert, but rethrow if it does

    `removeUserPermission(int $userId, int $pageId): bool`:
    - Use Capsule::table('user_permissions')->where('user_id', $userId)->where('page_id', $pageId)->delete()
    - Return (bool) ($affected > 0)

    `getUserMatrix(int $userId, int $levelId): array`:
    - SELECT from pages p, LEFT JOIN level_permissions lp ON lp.page_id = p.id AND lp.level_id = $levelId, LEFT JOIN user_permissions up ON up.page_id = p.id AND up.user_id = $userId
    - WHERE p.is_active = true
    - selectRaw with CASE: WHEN up.is_granted = false THEN false, WHEN up.is_granted = true THEN true, WHEN lp.page_id IS NOT NULL THEN true, ELSE false END AS has_access
    - ORDER BY p.urutan_tampil
    - Map to array: ['id' => (int), 'nama_page' => string, 'route_path' => string, 'has_access' => (bool)]

    Constructor: inject `PermissionRepositoryInterface $permissionRepo` and `UserRepositoryInterface $userRepository` as private readonly promoted properties.

    `__invoke(Request $request, Response $response, array $args): Response` — dispatch by HTTP method:
    - GET → $this->getMatrix($request, $response, (int) $args['userId'])
    - POST → $this->assign($request, $response, (int) $args['userId'])
    - DELETE → $this->remove($request, $response, (int) $args['userId'])
    - default → errorResponse 405 METHOD_NOT_ALLOWED

    `assign()` method — grants or denies based on is_granted field in body:
    1. Validate user exists via $this->userRepository->findById($userId) → return notFoundResponse('User not found') if null
    2. Parse body, validate page_id is present and is integer → validationErrorResponse if missing
    3. Validate is_granted is present and is boolean → validationErrorResponse if missing or invalid (use filter_var with FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE — if null, return validation error)
    4. Call $this->permissionRepo->assignUserPermission($userId, $pageId, $isGranted)
    5. Catch NotFoundException → return notFoundResponse('Page not found')
    6. Return jsonResponse with ['message' => $isGranted ? 'Permission granted' : 'Permission denied'], status 201

    `remove()` method:
    1. Validate user exists → notFoundResponse if null
    2. Parse body, validate page_id is present → validationErrorResponse if missing
    3. Call $this->permissionRepo->removeUserPermission($userId, $pageId)
    4. If returns false → return notFoundResponse('Permission override not found')
    5. Return jsonResponse with ['message' => 'Permission override removed'], status 200

    `getMatrix()` method:
    1. Validate user exists via userRepository → notFoundResponse if null
    2. Get user's level_id from the user entity ($user->levelId)
    3. Call $this->permissionRepo->getUserMatrix($userId, $levelId)
    4. Return jsonResponse with ['data' => $matrix], status 200
